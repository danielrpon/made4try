# 📈 TCX to XLSX Converter

Convierte archivos de entrenamiento **.TCX** (o comprimidos en `.tcx.gz`) en hojas de cálculo **.XLSX limpias**, listas para analizar en Excel o Google Sheets.  

Incluye automáticamente:
- **EF (Efficiency Factor)** = Potencia / Frecuencia cardíaca.  
- **DA (Desacople Aeróbico)** = Diferencia de EF con respecto al registro anterior.
